Giới thiệu công cụ

Công cụ chia tách file PDF miễn phí, hoạt động hoàn toàn trên trình duyệt. Cho phép xem trước từng trang, chọn trang cần giữ để tải xuống hoặc xóa bớt trang không cần thiết. File PDF không rời khỏi thiết bị, bảo mật tuyệt đối.

Xử lý nhanh

Tách trang PDF ngay trên trình duyệt, không cần cài đặt phần mềm hay upload lên máy chủ.

Riêng tư tuyệt đối

File PDF không rời khỏi thiết bị. Không server, không lưu trữ đám mây.

Xem trước từng trang

Hiển thị thumbnail và preview từng trang. Hỗ trợ chế độ xem danh sách và lưới.

Cách sử dụng

1

Chọn file PDF

Kéo thả hoặc bấm nút để chọn file PDF từ thiết bị của bạn.

2

Chọn trang cần tách

Đánh dấu checkbox các trang muốn giữ lại. Bấm vào trang để xem trước nội dung (chế độ danh sách).

3

Tải về hoặc xóa trang

Bấm "Tải trang đã chọn" để tải file PDF mới chỉ gồm các trang đã chọn, hoặc "Xóa trang đã chọn" để loại bỏ chúng khỏi file.

Câu hỏi thường gặp

File PDF có được upload lên server không?

Không. Toàn bộ quá trình xử lý diễn ra trên trình duyệt, file không bao giờ rời khỏi thiết bị.

Tôi có thể tách nhiều trang cùng lúc không?

Có. Bạn có thể chọn nhiều trang bằng checkbox, sau đó tải xuống tất cả trong một file PDF duy nhất.

Xóa trang có ảnh hưởng file gốc không?

Không. File gốc trên máy bạn không bị thay đổi. Thao tác xóa chỉ tác động lên bản sao đang hiển thị trên trình duyệt.

Có giới hạn kích thước file không?

Công cụ hỗ trợ file PDF tối đa 200 MB. File lớn có thể xử lý chậm hơn trên thiết bị cấu hình thấp.
